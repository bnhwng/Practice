Attributes summary

HTMl comes with attributes including width height to set the w and h of the photo or video according to the resolution without the need of css styling href attribute to attach link and src to attach video/image source. Style attribute can be used to quickly set the style such as color, font, size and more. The lang attribute that set the entire html document to the set language of the website. The title attribute is used as a tooltip to show additional information regarding the element in the html

All HTML elements can have attributes The href attribute of specifies the URL of the page the link goes to The src attribute of specifies the path to the image to be displayed The width and height attributes of provide size information for images The alt attribute of provides an alternate text for an image The style attribute is used to add styles to an element, such as color, font, size, and more The lang attribute of the tag declares the language of the Web page The title attribute defines some extra information about an element

HTML styles checkpoint

html style attribute have the following syntax "tagname style="property:value;>
The property is a css property, value is css value.

The css background-color property defines the background color of the element. For instance, this paragraph used background-color: light-blue, width: 70%;

CSS property color: defines the text color of the HTML element. Syntax style="color: red"; defines this paragraph style.

CSS font-family property defines the font of the element. The syntax style="font-family: font style" defines this paragraph font.

The css property text-align is used to defines the horizontal text alignment for the HTMl element. The syntax style="background-color: lightcoral; color: white; font-family: fontstyle; width: 70%; text-align: center" defines this HTML element style. 

Summary: Use the style attribute for styling HTML elements
         Use background-color for background color
         Use color for text colors
         Use font-family for text fonts
         Use font-size for text sizes
         Use text-align for text alignment.
