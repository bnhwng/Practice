HTML Blocks

Every HTML element has a default display value, depending on what type of element it is. Two most common display values are block and inline

  • A block level element starts on a new line, and the browser automatically add some space (margin) before and after the element.
  • A block level element always take the full space that it was declared in. For instance if declared in a global environment, it will stretchs to the most left and most right.
  • Two commonly used block elements are p and div.
    • The p element defines a paragraph in an HTML document.
    • The div element defines a division or a section in an HTML document.

HTML Inline Elements

  • An inline element does not start on a new line
  • An inline element only takes up as much width as necessary.
