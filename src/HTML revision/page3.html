HTML Images

Syntax: imgopentag src="src" alt="altdescript" closetag
      
Img tag is empty, similar to hr and br tag does not need a closing tag
      
You can add width and height attribute to an image to preset the size of the image.
      
There are several ways to use an image:
      
  *Use image just as image :D
      
  *Use image as a link by putting image inside an a tag
      
Stying:
      
  *Use width and height as mentioned.
      
  *Use float css property to let image float to the right or left of a text. Syntax float:left or float:right

Table Border

Table border css property can be used to manipulate the look of HTML table
      
  *Syntax: border: thicknesspx table-border-attribute color;
      
  *Rounded table: can be styled using border-radius property. Syntax border-radius: amount-of-px-to-be-curved-px;
      
  *The solid as default syntax can be changed to the following, looks as keywords: dotted, dashed, double, groove, ridge, inset, outset, none or hidden.
      
  *Border-color property can be used to change the border color.

Table Headers

This section regards the syntax of table headers: table headers are defined with th elements Each th elements represents a table cell.

Example for horizontal table headers:

First Name Last Name DOB
Ngoc Hung Bui 15-02-2001
Y Nhi Mai 26-03-2001

Example for vertical table headers:

First Name Ngoc Hung Y Nhi
Last Name Bui Mai
Age 23 23

Example for multiple column headers

We use colspan attribute for th element. Here we used colspan="2" for Full Name th element

Full Name Age
Ngoc Hung Bui 23
Y Nhi Mai 23
