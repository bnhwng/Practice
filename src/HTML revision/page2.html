HTML Styles - CSS

CSS stands for Cascading Style Sheets. CSS helps us control the layout of multiple wb pages all at once. And it is used to style the element in HMTL
    
Css is used to control the layout of a webpage. With CSS you can control the font, the size of the text and the spaces between elements. How elements are positioned and laidout.
    
There are three ways to use css in html: 
    
Inline by using style attribute as I have been using for the past two pages.
    
Internal by using a style element in the HTML head seciton
    
External by creating an additional CSS file. link by using link rel="stylesheet" href=""
    
    
Text properties in CSS:
    
* color: control the color of the text.
    
* font-family: control the font of the text.
    
* font-size: control the font size of the text.
    
=> For instance, this text style is color: blue; font-family: font; font-size: small; font-weight:bold;
